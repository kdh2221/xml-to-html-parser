================================================================
  참조 샘플 쌍 (Reference Pairs)
================================================================

절대좌표 원본과 상대좌표 변환 결과를 1:1 쌍으로 보관한다.
변환 엔진이 이 샘플들을 학습하여 패턴을 자동 추출한다.

파일명 규칙:
----------------------------------------------------------------
  {파일명}.xml         — 절대좌표 원본 XML
  {파일명}_pub.xml     — 상대좌표 변환 완료 XML

예시:
  화면조회.xml
  화면조회_pub.xml

주의사항:
----------------------------------------------------------------
- 반드시 원본 / _pub 쌍으로 넣을 것
- 파일명이 동일해야 매칭됨 (화면조회.xml ↔ 화면조회_pub.xml)
- 잘 만들어진 샘플일수록 변환 품질이 높아짐
================================================================
